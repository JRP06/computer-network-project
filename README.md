# Computer Network Project
- This is a class project for my course CSCE 3530 at UNT. 
- This project consists of building a virtual small-office network built in GNS3 demonstrating DHCP, DNS, ARP, HTTP/HTTPS, NAT, TCP/UDP, and SSL/TLS using Wireshark packet analysis.

# Devices 
- Router -NAT/Gateway - 192.168.1.1
- Switch - Layer 2 backbone
- DHCP/DNS Server – Name resolution + IP assignment - 192.168.1.2
- Web Server - HTTP/HTTPS - 192.168.1.3
- Client PC1 - DHCP client - 192.168.1.10
- Client PC2 - HDCP client – 192.168.1.11
- Monitoring Node – Wireshark/Scapy capture – 192.168.1.99
